# To-Do List Manager

A simple desktop to-do list app with a graphical interface built using Python and Tkinter.

## Features
- Add new tasks
- Remove tasks by selecting them
- Mark tasks as done
- Tasks automatically save to file
- Tasks persist between sessions

## How to Use
1. Type a task in the text box and click "Add"
2. Click on any task to select it
3. Click "Mark Done" to mark selected task as complete
4. Click "Remove" to delete selected task
5. All changes are automatically saved

## Requirements
- Python 3.x
- Tkinter (usually comes with Python)
